## 发现并修复的问题

### 🔴 Bug 1：安全漏洞（已修复 ✅）
- **问题**：`if (authToken) { 检查额度 }` 逻辑错误——没 Token 时跳过检查，直接访问 AI
- **影响**：任何人无需注册可直接调用 AI 接口，无限使用
- **修复**：改为 `if (!authToken) { return 401 }` 先拒绝，再检查额度
- **验证**：无 Token 请求现在返回 `{'error': 'Please verify your email first'}`

### 🔴 Bug 2：system prompt 未注入（已修复 ✅）
- **问题**：前端传 `system + messages` 格式，但后端只用 `messages`，忽略 `system`
- **影响**：AI 不带角色设定生成内容，质量不稳定
- **修复**：检测到 `system` 时，注入为第一条 system 消息
